# Real-estate-Price-Prediction-Analysis
Real estate Price Prediction Analysis

Project Overview
The Real Estate Price Prediction Analysis project aims to predict property prices using historical real estate data. By applying data analysis and machine learning techniques, the project identifies key factors that influence housing prices such as location, size, number of rooms, amenities, and market trends

Project Description
This project involves:

Cleaning raw realestate data by removing inconsistencies, duplicates, and formatting errors.

Creating structured datasets to improve readability and usability.

Using Pivot Tables and Charts to analyze realestate information such as:
City-Wise distribution
Area Wise - Price Distribution

Key Features
✔ Data Cleaning & Preparation
✔ Excel-based Data Analysis
✔ Pivot Table summarization
✔ Interactive Pivot Charts for visualization
✔ Insight extraction for Seller and Buyer decision-making
